# Baibai Guochan LLM v0.4.5

This release cleans up text-mode Agnet tool protocol output in the desktop transcript.

## Changes

- Hid raw `[Tool Call id=...]` markers from assistant messages while keeping the visible prose around them.
- Hid raw `[Tool Result for ...]` payloads so tool output appears through the normal tool cards instead of large markdown bubbles.
- Hid synthetic interruption placeholders such as `[Request interrupted by user]` from chat history.
- Applied the cleanup to restored transcript history, live streaming output, final render fallback, and non-streamed server message translation.
- Preserved ordinary assistant message ids and line endings when no tool protocol text is present.
