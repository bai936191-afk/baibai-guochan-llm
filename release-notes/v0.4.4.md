# 白白国产大模型 v0.4.4

This release focuses on packaging alignment and stronger Agnet tool compatibility.

## Changes

- Hardened built-in Agnet-compatible tools so schema or parameter mismatches return a soft tool result instead of surfacing raw `InputValidationError` messages to the model.
- Added compatibility aliases for common model-emitted tool fields, including file paths, URLs, commands, note bodies, and HTTP headers.
- Made file editing more tolerant of line-ending and whitespace differences, reducing brittle replace failures during code changes.
- Aligned desktop release artifact names with the documented `Baibai-Guochan-LLM` download names while keeping the installed application name as `白白国产大模型`.
- Updated release workflow validation and release metadata for the `v0.4.4` package set.
